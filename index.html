পি আই আর মোশন সেন্সর দিয়ে মোশন ডিটেকশন

আজ এই টিউটোরিয়ালে আমরা দেখব কিভাবে পি আই আর মোশন সেন্সর দিয়ে কোন বস্তু বা ব্যাক্তির মোশন বা সহজ ভাষায় নড়াচড়া ডিটেক্ট করা যায়, এবং এর ব্যাবহার কোথায় করা যেতে পারে তা জানব। আগেই বলে নিচ্ছি শুনতে যতই কঠিন মনে হোক না কেন এটি মাইক্রোকন্ট্রোলার দিয়ে করা অনেকগুলো সহজ প্রজেক্টের মধ্যে একটি। তুমি যদি মাইক্রোকন্ট্রোলার দিয়ে একটি এলইডি বাতি জ্বালাতে নিভাতে পার তাহলে এটাও তোমার কাছে কিছুই মনে হবে না।

পি আই আর (PIR) মোশন সেন্সর :

এই মজার কিন্তু উপকারী প্রজেক্টটি শুরু করার আগে চল পি আই আর মোশন সেন্সর সম্পর্কে জেনে নেয়া যাক। পি আই আর (PIR) এর পুরো নাম হচ্ছে প্যাসিভ ইনফ্রারেড। ইনফ্রারেড নামটি শুনে পরিচিত মনে হচ্ছে? হতেই পারে। ইনফ্রারেড বা অবলোহিত হচ্ছে আলোর একটি রূপ যা আমরা খালি চোখে দেখতে পাই না। কাজেই বুঝতেই পারছ পি আই আর মোশন সেন্সর মূলত আলোর বৈশিষ্ট্য ব্যবহার করে মোশন ডিটেক্ট করে থাকে। তোমার কি মনে কেন একে প্যাসিভ ইনফ্রারেড বলা হয়? তার আগে জেনে নেই কিভাবে এটি কাজ করে। তাহলে বুঝতে পারবে কেন একটিভ না হয়ে প্যাসিভ ইনফ্রারেড বলা হয়ে থাকে।

পি আই আর সেন্সর কোন বস্তু অথবা প্রাণী দ্বারা বিকির্ন ইনফ্রারেড রশ্মি থেকে মোশন ডিটেক্ট করে থাকে। স্থির কোন বস্তু বা ব্যাকটি থেকে একটি নির্দিষ্ট পরিমাণ অবলোহিত আলো বের হয়। যখন ব্যাক্তি বা বস্তুতই নড়া শুরু করে তখন এই অবলোহিত রশ্মিতে পরিবর্তন আসে। আর এই সামান্য পরিবর্তন কাজে লাগিয়ে পি আই আর সেন্সর কোন ব্যাকটি বা বস্তুর গতিবিধি নির্নয় করতে পারে। যেহেতু সেন্সরটি নিজে আলোর উৎস না বরং অন্য উৎস থেকে আসা আলো দিয়ে কাজ করে তাই একে প্যাসিভ বা পরোক্ষ বলা হয়ে থাকে।

বিঃদ্রঃ- পি আই আর সেন্সর সম্পর্কে আরও জানতে এই লিঙ্ক টি ঘুরে আসতে পার।

পি আই আর মোশন সেন্সর সম্পর্কে টেকনিকাল ডিস্ক্রিপশন:

আমাদের এই সেন্সরটিতে মোট তিনটি পিন আছে; VCC, GND আর Output. অন্য সব ডিজিটাল ডিভাইসের মত এই সেন্সরের VCC আর GND পিন যথাক্রমে 5V(dc) এবং 0V এর সাথে কানেক্ট করা লাগবে। আর Output পিনটিকে মাইক্রোকন্ট্রোলারের যে পিনে আমরা রীড করতে চাই সে পিনে কানেক্ট করব। মনে রাখা লাগবে অন্য অনেক সেন্সরের মত আমাদের পি আই আর সেন্সর অ্যানালগ আউটপুট দেয় না, ডিজিটাল (হাই/লো)আউটপুট দেয়। কাজেই এই প্রজেক্টে এডিসি পিনের কোন ব্যবহার নেই, যা আমাদের এই প্রজেক্টটিকে সহজ করে দেয়। এছাড়াও এই সেন্সরে রয়েছে দুইটি ভেরিএবল রেসিস্টর(পট) যেগুলোর মান পরিবর্তনের মাধ্যমে সেন্সরের সেন্সিটিভিটি এবং ডিলে এডজাস্ট করা যায়।

প্রয়োজনীয় কোডঃ

AVR Code

                  
#include< avr/io.h >
#define F_CPU 8000000
#include< util/delay.h >
int main()
{
	DDRA=0x00; //configuring PORTA as input
	PORTA=0x00;
	DDRB=0x08; // configuring PortB pin3 as output
	PORTB=0x00; // buzzer/LED off

	_delay_ms(10000);  //setting 10 seconds delay for sensor initialization

  while(1) {         //infinite loop
    if((PORTA & (1 << 0)) == 1) {    //condition for sensor output
       PORTB = 0xff;      //setting PORTB value to 1, buzzer and LED on
       Delay_ms(5000);    //buzzer and LED on for 5 seconds
       PORTB = 0x00;      //PORTB again 0, buzzer and LED off
     }
     else {
       PORTB = 0x00;     //buzzer and LED off
     }
}

PIC Code

                  
void main() {
  TRISB = 0xff;    //declaring PORTB as sensor input
  PORTB = 0x00;    //asigning value 0 to PORTB
  TRISC = 0x00;    //declaring PORTC as sensor output
  PORTC = 0x00;    //asigning value 0 to PORTC, buzzer and LED off

  Delay_ms(10000);  //setting 10 seconds delay for sensor initialization

  while(1) {         //infinite loop
    if((PORTB & (1 << 0)) == 1) {    //condition for sensor output
       PORTC = 0xff;      //setting PORTC value to 1, buzzer and LED on
       Delay_ms(5000);    //buzzer and LED on for 5 seconds
       PORTC = 0x00;      //PORTC again 0, buzzer and LED off
     }
     else {
       PORTC = 0x00;     //buzzer and LED off
     }
   }
 }
